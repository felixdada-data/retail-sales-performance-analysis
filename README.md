# Retail Sales Performance Analysis

## Project Overview
This project analyses retail sales data to identify sales trends, top performing regions, product categories, sales representatives, and payment status.

## Business Questions
1. Which region generated the highest sales?
2. Which product category produced the most revenue?
3. Which sales representative performed best?
4. What proportion of sales were paid, pending, or refunded?

## Tools Used
- Microsoft Excel
- MySQL Workbench
- Tableau Public
- Power BI

## Data Cleaning Process
- Removed duplicate records
- Standardised region names
- Standardised product category names
- Removed incomplete records
- Removed invalid negative sales values
- Created a Sales Total column

## Key Findings
- West generated the highest sales revenue.
- Electronics was the top product category.
- Daniel was the highest performing sales representative.
- Most sales were paid, while some remained pending or refunded.

## Business Recommendations
- Study the West region sales strategy and apply useful practices in lower performing regions.
- Increase stock and promotion for high performing electronics products.
- Follow up on pending payments to improve cash collection.
- Support lower performing sales representatives with targeted coaching.
